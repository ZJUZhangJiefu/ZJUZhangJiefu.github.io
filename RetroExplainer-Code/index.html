    4.2.3 data/data_utils.py/map_id_alignment函数
    • 函数参数
      • reactant_smiles, product_smiles：即反应物与产物的SMILES字符串序列。
      • product：用于更新原子映射编号的参考产物。这一参数在重新排列已有的逆合成预测结果时，才会被使用。

        def map_id_alignment(reactant_smiles, product_smiles, product=None, return_smiles=True):
    """
    if product is None, update product mapping numbers according to canonical atom order (for USPTO-50K)
    else, update product mapping numbers according to the reference product (for re-ranking dataset)
    :param reactant_smiles:
    :param product_smiles:
    :param product:
    :param return_smiles:
    :return:
    """
    • 原子映射序号调整
      • 下列变量中，cano是标准原子顺序(canonical atom order)的缩写。
      • 首先，cur_product是原子映射序号未经调整的、rdkit.Chem.Mol类型产物。
      • index2maxnums字典，用于查询原子序号(index)到原子映射序号(atom map number)的对应关系。
      • 若参考产物(product)传入类型为SMILES字符串，则调用Chem.MolFromSmiles读取，作为设置原子映射序号的参考Mol对象mol_cano；若为Mol类型，则调用Chem.RWMol函数读取；若product为空，则令当前产物cur_product的所有原子映射序号均为0，重新化为SMILES字符串后，再次转化为Mol实例。
        index2mapnums = {}
cur_product = Chem.MolFromSmiles(product_smiles)
for atom in cur_product.GetAtoms():
    index2mapnums[atom.GetIdx()] = atom.GetAtomMapNum()

if isinstance(product, str):
    mol_cano = Chem.MolFromSmiles(product)

elif product is None:
    mol_cano = Chem.RWMol(cur_product)
    [atom.SetAtomMapNum(0) for atom in mol_cano.GetAtoms()]
    smi_cano = Chem.MolToSmiles(mol_cano)
    mol_cano = Chem.MolFromSmiles(smi_cano)

else:
    mol_cano = Chem.RWMol(product)
    • 函数GetSubstructMatches的描述：Returns the indices of the molecule’s atoms that match a substructure query. （链接）

    • matches为tuple类型。

    • new_reactant首先将反应物的原始SMILES（有信息泄露）读取为Mol实例，然后更改反应物的原子映射序号。
    • 下面使用产物的旧序号->新序号对应关系更改反应物的原子映射序号，没有太大的道理可言，只是实现外部参数reshuffle打乱AtomicMapNum，从而防止信息泄露的目的。

# print(matches)

    for atom, mat in zip(mol_cano.GetAtoms(), matches[0]):

            # to completely remove potential information leak
            mapnums_old2new[index2mapnums[mat]] = 1 + atom.GetIdx()
            atom.SetAtomMapNum(1 + atom.GetIdx())
        else:
            mapnums_old2new[index2mapnums[mat]] = atom.GetAtomMapNum()
    new_product = mol_cano
    # update reactant mapping numbers accordingly

    new_reactant = Chem.MolFromSmiles(reactant_smiles)
    for atom in new_reactant.GetAtoms():
        if atom.GetAtomMapNum() in mapnums_old2new.keys():
            atom.SetAtomMapNum(mapnums_old2new[atom.GetAtomMapNum()])
        else:
            atom.SetAtomMapNum(0)

    if product is not None:
        assert Chem.MolToSmiles(new_product) == Chem.MolToSmiles(product), \
            f"reactant_smiles:{reactant_smiles}, product_smiles:{product_smiles}, " \
            f"product:{Chem.MolToSmiles(product)}, new_product:{new_product}, " \
            f"new_reactant:{Chem.MolToSmiles(new_reactant)}"

    if return_smiles:
        return Chem.MolToSmiles(new_reactant), Chem.MolToSmiles(new_product)
    else:
        return new_reactant, new_product

        def get_bond_adj(mol):
    """
    :param mol: rdkit mol
    :return: multi graph for {
                sigmoid_bond_graph,
                pi_bond_graph,
                2pi_bond_graph,
                aromic_graph,
                conjugate_graph,
                ring_graph,
    }
    """
    n_atom = len(mol.GetAtoms())
    bond_adj = torch.zeros(n_atom, n_atom, dtype=torch.uint8)
    for bond in mol.GetBonds():
        i, j = bond.GetBeginAtomIdx(), bond.GetEndAtomIdx()
        bond_type = bond.GetBondTypeAsDouble()
        bond_adj[i, j] = bond_adj[j, i] = 1
        if bond_type in [2, 3]:
            bond_adj[i, j] = bond_adj[j, i] = bond_adj[i, j] + (1 << 1)
        if bond_type == 3:
            bond_adj[i, j] = bond_adj[j, i] = bond_adj[i, j] + (1 << 2)
        if bond_type == 1.5:
            bond_adj[i, j] = bond_adj[j, i] = bond_adj[i, j] + (1 << 3)
        if bond.GetIsConjugated():
            bond_adj[i, j] = bond_adj[j, i] = bond_adj[i, j] + (1 << 4)
        if bond.IsInRing():
            bond_adj[i, j] = bond_adj[j, i] = bond_adj[i, j] + (1 << 5)
    return bond_adj

    (一)特征嵌入表示

    1.原子特征高斯变换层
    • 模块作用：对原子特征进行高斯变换。
    • 初始化参数
      • 调用nn.Embedding(num_embeddings, embed_dim)对means, stds, mul, bias进行初始化。
      • nn.Embedding(num_embeddings, embed_dim)作用
        • 本质：查找表，输入token的索引，输出token的嵌入表示(embeddings)。
        • 嵌入表示：可以在模型训练过程中不断更新。
        • num_embeddings：表示词汇表(vocabulary)的总token数。
        • embed_dim：表示每个token进行嵌入表示后的维度数目。
      • 此处，means, stds两个嵌入表示的vocabulary size = 1，嵌入表示维度为128；mul, bias的vocabulary size = 1，嵌入表示维度为1。
      • 调用nn.init.uniform_，从均匀分布中随机采样，初始化means, stds的嵌入表示参数；用常数初始化mul, bias的嵌入表示参数。
    • forward计算过程
      • 输入x：由若干原子特征形成的张量，形状为[batch_size, 300]。
      • 首先调用unsqueeze(-1)，形状：[batch_size, 300] -> [batch_size, 300, 1]。
      • x = mul * x.unsqueeze(-1) + bias：对x进行线性变换。
      • x = x.expand(-1, -1, self.d_model)的作用：扩张张量x的维度，其中expand函数的各参数，分别对应于扩张后各维度的大小。作用前，x.shape = [batch_size, 300, 1]，expand(-1, -1, self.d_model)表示第0, 1维度的size不变，self.d_model表示第2维度的size从1变为128。
      • expand函数扩张维度的本质：只是基于同一片内存，创建多个视图，实质上并不进行数据拷贝和内存空间扩张。比如此处的x.shape: [batch_size, 300, 1] -> [batch_size, 300, self.d_model]，对于x[0][0][0]这么一个浮点数，就创造了self.d_model个视图，这些新视图完全对应于同一个浮点数的内存空间，也就是x[0][0][0] ~ x[0][0][self.d_model - 1]，全部都对应于x[0][0][0]这一浮点数的内存空间。
      • 最后使用mean, std两个参数，返回x归一化后，输入到高斯分布函数获得的结果。

    • 高斯函数

      def gaussian(x, mean, std): 
    return torch.exp(-0.5 * (((x - mean) / std) ** 2)) / (A * std)

    • 上述函数对应公式：\(G(x) = \frac{1}{\sqrt{2\pi} \sigma}e^{- \frac{1}{2}(\frac{x - \mu}{\sigma})^2}\)。

    model/Embeddings.py
    class GaussianAtomLayer(nn.Module):
    def __init__(self, d_model=128, means=(0, 3), stds=(0.1, 10)):
        super().__init__()
        self.d_model = d_model
        self.means = nn.Embedding(1, d_model)
        self.stds = nn.Embedding(1, d_model)
        self.mul = nn.Embedding(1, 1)
        self.bias = nn.Embedding(1, 1)
        nn.init.uniform_(self.means.weight, means[0], means[1])
        nn.init.uniform_(self.stds.weight, stds[0], stds[1])
        nn.init.constant_(self.bias.weight, 0)
        nn.init.constant_(self.mul.weight, 1)

    def real_forward(self, x):
        mul = self.mul.weight
        bias = self.bias.weight
        x = mul * x.unsqueeze(-1) + bias # [batch_size, 300] -> [batch_size, 300, 1]
        x = x.expand(-1, -1, self.d_model) # [batch_size, 300, 1] -> [batch_size, 300, d_model]
        mean = self.means.weight[0]
        std = self.stds.weight[0].abs() + 1e-5
        return gaussian(x, mean, std).type_as(self.means.weight)

    def forward(self, x):
        """
        :param x: [bsz, n_atom]
        :return: [bsz, n_atom, d_model]
        """
        out = self.real_forward(x)
        return torch.where(x.unsqueeze(-1).expand_as(out) != 0, out, torch.zeros_like(out))

      • 对于线性层、嵌入表示层，分别按照不同参数的正态分布，对参数进行采样，以实现初始化目的。

      • 输入特征：atom_fea，维度为[batch_size, n_fea_type（特征类型数）, n_atom（单个产物的最大原子数）]，进行代码调试时，atom_fea.shape实际为[128, 9, 300]。

      • 9种原子特征：使用加性方法进行汇聚。
      • graph_token特征的汇聚
        • self.graph_token是一个nn.Embedding(1, 128)的组件，故：
          self.graph_token.weight.shape = [1, 128]
          self.graph_token.weight[0].shape = [128]
          graph_token.shape = [128]
        • if contrast：使用加性机制，汇聚graph_token和作为对比的contrast_token，这两个token都是全局范围的特征。

        • 图特征、反应类型嵌入表示、中心个数嵌入表示，相加进行汇聚。
      • 特征输出
        • atom_fea -> out是原子（细粒度）特征的嵌入表示，graph_token是图（粗粒度）特征的嵌入表示，二者的形状分别为[bsz, 300, 128], [bsz, 1, 128]。

        • 输出特征out是粗、细粒度特征的结合。

    class AtomFeaEmbedding(nn.Module):
    def __init__(self, d_model, atom_dim=65, total_degree=20, formal_charge=8, hybrid=10, exp_valance=8,
                 hydrogen=8, aromatic=2, ring=9, n_layers=1, need_graph_token=True, known_rxn_type=True,
                 known_rxn_cnt=True, n_rxn_cnt=10):
        super(AtomFeaEmbedding, self).__init__()
        self.known_rxn_cnt = known_rxn_cnt
        self.known_rxn_type = known_rxn_type

        self.atom_encoders = nn.ModuleList([
            nn.Embedding(atom_dim + 1, d_model, padding_idx=0),  # 65
            nn.Embedding(total_degree + 1, d_model, padding_idx=0),  # 20
            nn.Embedding(hybrid + 1, d_model, padding_idx=0),  # 10
            nn.Embedding(hydrogen + 1, d_model, padding_idx=0),  # 10
            nn.Embedding(aromatic + 1, d_model, padding_idx=0),  # 2
            nn.Embedding(ring + 1, d_model, padding_idx=0),  # 9
            nn.Embedding(30, d_model, padding_idx=0),
            nn.Embedding(5, d_model, padding_idx=0),
            GaussianAtomLayer(d_model, means=(-1, 1), stds=(0.1, 10))
        ])
        self.need_graph_token = need_graph_token
        # self.feed_forward = nn.Sequential(
        #     nn.LayerNorm(d_model),
        # )

        if need_graph_token:
            self.graph_token = nn.Embedding(1, d_model)
            if known_rxn_cnt:
                self.cnt_token = nn.Embedding(n_rxn_cnt, d_model)
            if known_rxn_type:
                self.type_token = nn.Embedding(10, d_model)
            self.contrast_token = nn.Embedding(1, d_model)

        self.apply(lambda module: init_params(module, n_layers=n_layers))

    def forward(self, atom_fea, center_cnt=None, rxn_type=None, contrast=False):
        """
        :param atom_fea: [bsz, n_fea_type, n_atom]
        :return: [bsz, n_atom + I, d_model] I = 1 if need_graph_token else 0
        """
        bsz, n_fea_type, n_atom = atom_fea.size()
        out = self.atom_encoders[-1](atom_fea[:, -1])

        for idx in range(n_fea_type - 1):
            # print(idx, atom_fea[:, idx].max().item())
            # print(idx, self.atom_encoders[idx].weight.size()[0], atom_fea[:, idx].max().int().item())
            out += self.atom_encoders[idx](atom_fea[:, idx].int())

        if self.need_graph_token:
            graph_token = self.graph_token.weight[0].clone()
            if contrast:
                graph_token += self.contrast_token.weight[0]

            graph_token = graph_token.view(1, 1, -1).repeat(bsz, 1, 1)

            if self.known_rxn_type:
                graph_token += self.type_token(rxn_type).view(bsz, 1, -1)
            if self.known_rxn_cnt:
                graph_token += self.cnt_token(center_cnt).view(bsz, 1, -1)

            out = torch.cat([graph_token, out], dim=1)

        return out

    class GaussianBondLayer(nn.Module):
    def __init__(self, nhead=16, means=(0, 3), stds=(0.1, 10)):
        super().__init__()
        self.nhead = nhead
        self.means = nn.Embedding(1, nhead)
        self.stds = nn.Embedding(1, nhead)
        self.mul = nn.Embedding(1, 1)
        self.bias = nn.Embedding(1, 1)
        nn.init.uniform_(self.means.weight, means[0], means[1])
        nn.init.uniform_(self.stds.weight, stds[0], stds[1])
        nn.init.constant_(self.bias.weight, 0)
        nn.init.constant_(self.mul.weight, 1)

    def real_forward(self, x):
        mul = self.mul.weight
        bias = self.bias.weight
        x = mul * x.unsqueeze(-1) + bias
        x = x.expand(-1, -1, -1, self.nhead)
        mean = self.means.weight[0]
        std = self.stds.weight[0].abs() + 1e-5
        return gaussian(x, mean, std).type_as(self.means.weight)

    def forward(self, x):
        """
        :param x: [bsz, n_atom, n_atom]
        :return: [bsz, n_atom, n_atom, nhead]
        """
        out = self.real_forward(x)
        return torch.where(x.unsqueeze(-1).expand_as(out) != 0, out, torch.zeros_like(out))

      • n_graph_type：图的种类数，设定为6。
      • max_single_hop：
      • use_3d_info, use_dist_adj：是否使用三维信息、距离矩阵，均为布尔变量。
      • max_path：最大路径的长度。后续对边、邻域信息和原子环境(Atom Environment)进行嵌入表示时，对原子的最大邻域作出了限制；对于各原子的0 ~ max_path - 1-邻域信息，分别进行不同的嵌入表示。
      • head_dim：注意力头的维度，即总的嵌入表示维度/注意力头的个数。
      • self.edge_encoders：即max_path个边特征编码器，分别嵌入表示各原子的0 ~ max_path - 1-邻域信息。
      • self.spacial_encoder：使用高斯函数，编码原子的距离信息。
      • if need_graph_token代码段
        • 对于图级别的token、反应类型、反应个数，分别进行嵌入表示，均表示为n_heads维度，与attention head的个数相对应。
      • 最后调用init_params函数，进行组件的参数初始化，该函数之前已提及。
    • forward计算过程
      • 输入：bond_adj，shape = [batch_size, n_atoms, n_atoms]，此处设定为[128, 300, 300]。
      • comb_embed：即combined embedding的缩写，通过相加对各类嵌入表示进行汇总，初始化为0。两个if语句首先汇聚两种距离的嵌入表示信息。
      • comb_embed.shape = [bsz, n_atoms, n_atoms, n_heads]，设定为[128, 300, 300, 16]，与GaussianBondLayer的forward函数输出形状相同。
      • if self.max_single_hop > 0代码段
        • j_hop_embed：表示对于每个原子而言，恰好经过i条边，是否能够到达图中其他每一个原子，若是，则(j_hop_embed - 1) >> i在该位置的取值非0；若不是，则(j_hop_embed - 1) >> i在该位置的取值为0。
        • torch.where(condition, a, b)，对于张量的每一个元素，满足条件condition，则取值为a，否则取值为b。此处调用该函数，作用是筛选出图中各个原子的邻域信息。
        • i既表明邻域（恰好经过i步能到达的原子）距离，又是边特征编码器的索引。对于不同的图而言，同样的i用同一个edge_encoder编码，不同的i则使用不同的edge_encoder进行编码。
        • 各个edge_encoder的输出形状均为[128, 300, 300, 16]，直接加到comb_embed联合嵌入表示张量之上。
      • 经过permute(0, 3, 1, 2)，comb_embed的形状变化为：[128, 300, 300, 16] -> [128, 16, 300, 300]，这样做的目的是适应后续Transformer的自注意力机制。
      • 使用0.和-torch.inf，对于邻接矩阵bond_adj获得1-邻域信息的掩码mask。
      • if self.need_graph_token代码段
        • graph_token的形状为[16]，与attention heads的个数相对应。
        • graph_token和contrast_token通过相加进行汇聚。
        • self.type_token/cnt_token的形状为[128, 16]，经过view(bsz, self.num_heads, 1, 1)，形状变为[128, 16, 1, 1]。
        • expand函数对同一内存创建多个视图，使得graph_token的形状变化为：[128, 16, 1, 1] -> [128, 16, 300, 1]。将变换后的graph_token与comb_embed在最后一个维度进行concatenation，comb_embed的形状变化为：[128, 16, 300, 300] -> [128, 16, 300, 301]。即对于每个原子而言，原来是300维的细粒度嵌入表示，现在又分别增加了1维粗粒度的嵌入表示，因此每个原子的边特征（从connectivity出发的嵌入表示），同时包含了全局和局部两种信息。
        • 同样地，经过倒数第二个维度对graph_token与comb_embed特征的concatenation，comb_embed的形状变化为：[128, 16, 300, 300] -> [128, 16, 301, 301]，变换前，仅对每个原子计算全局、局部特征，变换后，专门增加了一些维度，用于全局特征的计算，让graph_token在特征计算时，与各个原子具有同等的地位。
        • mask的形状变化与comb_embed类似，变换前、第一个语句结束后、第二个语句结束后的形状分别为：[128, 300, 300] -> [128, 300, 301] -> [128, 301, 301]。
      • n_atom += 1，表示把graph_token视为一个与atom同等地位的token，与图神经网络中virtual node/super node具有一些相似的意义。
      • mask = mask.unsqueeze(1).expand(bsz, self.num_heads, n_atom, n_atom)：[128, 301, 301] -> [128, 1, 301, 301] -> [128, 16, 301, 301]，针对同一块内存空间创建多个视图。
      • 最后comb_embed与mask相加，意义是：comb_embed包含了不同邻域的嵌入表示信息，由于这种嵌入表示信息基于原子之间的k步可达性而定义，因此对于k步不可到达的原子对，它们之间的嵌入表示就没有意义了，用掩码使之被遮盖。
      • 最终输出维度：[128 * 16, 301, 301]。

    def __init__(self, embed_dim, n_head=16, max_paths=50, n_graph_type=6, max_single_hop=4, n_layers=1,
            need_graph_token=True, use_3d_info=False, known_rxn_type=True, known_rxn_cnt=True,
            use_dist_adj=True, n_rxn_cnt=10):
    """
    :param embed_dim:
    :param n_head:  n_head mast has to be 2 to the nth power
                    example: n_head=2^4, hop_distribution=[1, 2^3, 2^2, 2^1, 2^0]
                            n_head=2^n, hop_distribution=[1, 2^(n-1), 2^(n-2), ..., 2^0]
    :param max_paths:
    :param n_graph_type:
    :param max_single_hop:
    :param n_layers:
    """
    super().__init__()
    self.known_rxn_cnt = known_rxn_cnt
    self.known_rxn_type = known_rxn_type
    self.num_heads = n_head
    self.embed_dim = embed_dim
    # assert n_head & (n_head - 1) == 0, f"n_head mast has to be 2 to the nth power, but got{n_head}"
    self.n_graph_type = n_graph_type
    self.max_single_hop = max_single_hop
    self.use_3d_info = use_3d_info
    self.use_dist_adj = use_dist_adj
    # self.hop_distribution = [1 << i for i in range(int(math.log2(n_head) - 1), -1, -1)]
    # self.hop_distribution[-1] += 1
    # assert sum(self.hop_distribution) == self.num_heads
    self.max_paths = max_paths
    self.head_dim = embed_dim // n_head
    assert self.head_dim * n_head == self.embed_dim, "embed_dim must be divisible by num_heads"

    self.edge_encoders = nn.ModuleList([
        nn.Embedding(max_paths + 1, n_head, padding_idx=0) for _ in range(n_graph_type)
    ])
    self.spatial_encoder = GaussianBondLayer(n_head, means=(0, 3), stds=(0.1, 10))
    if self.use_3d_info:
        self.spatial3d_encoder = GaussianBondLayer(n_head, means=(0, 3), stds=(0.1, 10))

    # self.norm = nn.LayerNorm(n_head)

    self.need_graph_token = need_graph_token
    if need_graph_token:
        self.graph_token = nn.Embedding(1, n_head)
        if known_rxn_cnt:
            self.cnt_token = nn.Embedding(n_rxn_cnt, n_head)
        if known_rxn_type:
            self.type_token = nn.Embedding(10, n_head)
        self.contrast_token = nn.Embedding(1, n_head)

    self.apply(lambda module: init_params(module, n_layers=n_layers))
def forward(self, bond_adj, dist_adj, center_cnt=None, rxn_type=None, dist3d_adj=None, contrast=False):
        """
        :param bond_adj: [bsz, n_atom, n_atom]
        :param dist_adj: [bsz, n_atom, n_atom]
        :param rxn_type: [bsz]
        :return: attention bias with mask [bsz*n_head, n_atom, n_atom]
        """
        # bsz, n_hop, n_type, n_atom, _ = bond_adj.size()
        # bond_embed = self.edge_encoders[0](bond_adj[:, :, 0].int()).sum(dim=1)
        # for i in range(1, self.n_graph_type):
        #     bond_embed += self.edge_encoders[i](bond_adj[:, :, i].int()).sum(dim=1)

        # [bsz, n_hop, n_type, n_atom, n_atom, n_head] -> # [bsz, n_atom, n_atom, n_head]

        # bond_embed = torch.cat([
        #     bond_embed[:, i].unsqueeze(1).expand(bsz, hop, n_atom, n_atom)
        #     for i, hop in enumerate(self.hop_distribution)
        # ], dim=1)  # [bsz, n_hop, n_type, n_atom, n_atom] -> # [bsz, n_head, n_atom, n_atom]

        bsz, n_atom, _ = bond_adj.size()
        comb_embed = 0
        if self.use_dist_adj and dist_adj is not None:
            comb_embed += self.spatial_encoder(dist_adj)
        if self.use_3d_info and dist3d_adj is not None:
            comb_embed += self.spatial3d_encoder(dist3d_adj)

        if self.max_single_hop > 0:
            for i in range(self.n_graph_type):
                j_hop_embed = bond_adj.long()
                # decode to multi sense embedding
                j_hop_embed = torch.where(j_hop_embed > 0, ((j_hop_embed - 1) >> i) % 2, 0).float()
                base_hop_embed = j_hop_embed
                comb_embed += self.edge_encoders[i](j_hop_embed.int())
                for j in range(1, self.max_single_hop):
                    # # generate multi atom environment embedding
                    # j_hop_embed = torch.einsum('bnm,bjk->bnk', j_hop_embed, base_hop_embed)
                    j_hop_embed = torch.bmm(j_hop_embed, base_hop_embed)
                    j_hop_embed = self.max_paths - torch.relu(self.max_paths - j_hop_embed)
                    comb_embed += self.edge_encoders[i](j_hop_embed.int())

        comb_embed = comb_embed.permute(0, 3, 1, 2)
        mask = torch.where(bond_adj != 0, 0., -torch.inf)

        if self.need_graph_token:
            graph_token = self.graph_token.weight[0].clone()
            if contrast:
                graph_token += self.contrast_token.weight[0]
            graph_token = graph_token.view(1, self.num_heads, 1, 1).repeat(bsz, 1, 1, 1)

            if self.known_rxn_type:
                graph_token += self.type_token(rxn_type).view(bsz, self.num_heads, 1, 1)
            if self.known_rxn_cnt:
                graph_token += self.cnt_token(center_cnt).view(bsz, self.num_heads, 1, 1)

            comb_embed = torch.cat([graph_token.expand(bsz, self.num_heads, n_atom, 1),
                                    comb_embed], dim=-1)
            comb_embed = torch.cat([graph_token.expand(bsz, self.num_heads, 1, n_atom + 1),
                                    comb_embed], dim=-2)

            mask = torch.cat([torch.zeros_like(mask[:, :, 0]).unsqueeze(-1), mask], dim=-1)
            mask = torch.cat([torch.zeros_like(mask[:, 0, :]).unsqueeze(-2), mask], dim=-2)

            # [bsz, n_head, n_atom+1, n_atom+1]
            n_atom += 1

        mask = mask.unsqueeze(1).expand(bsz, self.num_heads, n_atom, n_atom)

        return (comb_embed + mask).reshape(bsz * self.num_heads, n_atom, n_atom)

      class RetroAGTEmbeddingLayer(nn.Module):
    def __init__(self, d_model, n_head, max_paths, n_graph_type, max_single_hop, atom_dim, total_degree,
                formal_charge, hybrid, exp_valance, hydrogen, aromatic, ring, n_layers, need_graph_token, n_rxn_cnt,
                use_3d_info=False, dropout=0., known_rxn_type=True, known_rxn_cnt=True, use_dist_adj=True):

        self.atom_encoder = AtomFeaEmbedding(d_model, atom_dim, total_degree, formal_charge, hybrid, exp_valance,
                                            hydrogen, aromatic, ring, n_layers, need_graph_token, known_rxn_type,
                                            known_rxn_cnt, n_rxn_cnt=n_rxn_cnt)

    def forward(self, atom_fea, bond_adj, dist_adj, center_cnt, rxn_type, dist3d_adj=None, contrast=False):

            self.edge_encoder(bond_adj, dist_adj, center_cnt, rxn_type, dist3d_adj, contrast)
